#Keyword Analysis
바닐라 javascript를 이용한 텍스트 분석 어플리케이션

## Setup

Install dependencies

```sh
$ yarn install (or npm install)
```

## Development

```sh
$ yarn dev (or npm run dev)
# visit http://localhost:8080
```

## Features

- 5000자 미만 영문 텍스트에 대한 분석 결과를 제공.
- 텍스트 수정시 분석 결과 업데이트.
- 단어를 기준으로 분석, 빈도수에 따른 데이터 시각화
